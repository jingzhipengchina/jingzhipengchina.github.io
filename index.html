Hexo博客搭建

关于Hexo博客搭建

Win10下的搭建

Node和Git的安装

首先需要下载相关的软件环境
Node.js和Git这两个工具

Node.js网址（Tips：下载可能有点慢，应该是不需要翻墙的）
https://nodejs.org
安装步骤非常简单，一直next，下一步就可以了，默认安装就行，当然，也可以自己选择安装位置，对后续没影响

Git
https://git-scm.com/downloads
然后我们选择windows版本的下载安装
也是一直点下一步，安装官方默认的来就行，
tips：这个Git Bash下载下来就相当于Linux中的终端窗口了，以后我们就用这个来打开终端，进行各种编辑和上传工作。

Hexo的安装

新建一个文件夹，比如我建了 ==myblog==
打开你的文件夹，然后在空白处点鼠标的右键，选择 ==Git Bash Here==

打开之后看看 ==node，npm==是否安装成功，没有成功的就重新安装node。
使用的命令分别是
node -v以及npm -v
安装成功的，他会弹出对应的版本号

针对npm在国内使用起来比较慢，我们需要使用国内镜像的npm工具cnpm来替代
安装cnpm提高速度，以后下载什么东西也都用cnpm
在上面终端继续输入
npm install -g cnpm –registry=https://registry.npm.taobao.org
测试cnpm-成功！

==完成之后安装hexo==
cnpm install -g hexo-cli

验证是否安装成功
hexo -v
出现,说明成功

首先看看当前路径命令pwd
结果为
Administrator@ZEPLYN WINDOW MINGW64 ~/d
然后我们
运行命令，这步是关键，主要是建立整个项目，这里我的文件夹名字是myblog，文件夹位置是桌面上。
hexo init

运行测试

常用的三个命令
hexo clean 用来清理缓存文件
hexo g 生成文件
hexo s 运行本地服务器
hexo d 上传至服务器

在输入hexo s后
就可以在本地浏览器输入 localhost：4000 查看
到此本地环境搭建完成

部署云端

在介绍了在本地搭建一个服务器并且成功运行测试，这次我们把它部署到云端，让别人也可以访问我们的网站，
我使用的是部署到github上，但是由于github服务器在国外，所以可能响应的速度有点慢
国内的话，有Coding可以替代,他已经和腾讯云合作了，速度应该还不错

此处以github为例

技术博客一

关于建立博客

Tips：
（写在开头）
作为一个微电子专业的工科生
我并没有接受正规的计算机方面的知识教育
所以以下所写都是个人自学所得
可能有很多错误的认识和疏漏
欢迎不吝赐教
我相信
能看下去的都是兴趣使然
欢迎讨论探究

步入正题

首先说说我眼中的博客吧
这个在当今时代或多或少有点落伍的东西
却依旧以他独特的魅力
广泛被用作技术和想法分享

现如今搭建博客最主流的要数WordPres了，
就现如今的大型公司网站也有很多是基于这个架构搭建的，
当然，他也属于开源的动态博客，
WordPress作为全球最大的开源网站搭建平台，
一直保持着极高的热度，，
当然开源不意味着完全免费，
想要搭建一个WordPress架构的博客，首先必须用到一台服务器，将自己的项目部署到这台服务器上，
虽然说，完全可以将自己的电脑当做服务器，但是，这需要电脑做到24小时开机，显然这很不切实际，
当然，不少人会想到用NAS来作为服务器，
这一点我到没有具体试过，因为我没有自己搭过NAS，更没有尝试过在NAS上部署WordPress，
不过，在和学校的一个创业老师聊天时，他有说自己尝试过NAS上搭建，不过遇到了问题，最终没有解决，最终还是选择了云服务器的方案
虽然说WordPress搭建博客这项技术已经很泛滥，
但是NAS上搭建还为数不多，
有兴趣的可以尝试，也欢迎探讨。
回到本题，继续说说WordPress搭站，
简单地说
他需要一台云服务器+一个域名，
一个搭建，一个部署，
操作起来倒是很简单，
因为前辈们早就发明了宝塔面板这种神奇的东西，
我在还不知道宝塔面板的时候尝试过自己手动敲代码搭建，
那叫一个烦啊，
虽然最终成功了，但是，经常出现问题，还会时不时地升级WordPress版本导致数据全部丢失，
所以宝塔面板搭建WordPress网站简直是把饭送到了嘴边。
上面说起来或许你感觉还不错，那我现在来算一算价格，
域名+云服务器
先说域名吧，本人倒是入手了两个域名，一个备案了，一个没有备案
WordPress搭建博客应该是要备案的，成本方面，一个很随意的域名可能很便宜，我入手的两个都是.xyz结尾的，几十块钱。
服务器嘛，国内用户大多会选择腾讯云和阿里云，他们学生是可以使用学生机的，我入手的是阿里云学生机，100多一年的样子，
对学生还是很优惠的，非学生的话，一个一般配置的那就要1000+了
国外就有很多了
耳熟能详的谷歌云，搬瓦工，Vulter，很多很多，不少也有优惠
他们的优点是可以用来部署科学上网，一机多用，还是值得入手的。

但是，对于我们这些学生来说，免费当然是最好的啦，有没有呢？
答案是肯定的

先来卖个关子
除了WordPress搭建博客之外，还可以用到静态博客框架
举栗子啦
Hexo
Jekyllrb
Gohugo
Vurpress
等等诸如此类
他们都有点CMS的味道

而我要说的就是Hexo这个架构
说到免费，当然少不了提一提Github这个大型同性交流平台啦
作为全世界开源项目的根据地
他一直勤勤勉勉，在被微软收购之后，他并没有改变它开源的本性
可以说是十分良心啦
Github实现免费的原因就在于
他会给你提供一个和域名+CES发挥相同作用的github.io结尾的地址
不仅可以将自己的东西存在上面，还可以让别人像域名那样访问，
简直良心
而这个免费的博客搭建方案就是才用的Hexo+Github
框架使用Hexo，部署在Github上
当然，如果对github.io结尾的域名没好感，
他也是可以再转到别域名上面的
其实之后的味道就和WordPress差不多了
你可以切换主题啊（github上有很多开源的hexo主题，都是大神做的）
写post博客啊，只不过，每次写完多要hexo d上传到github上
博文采用常规的markdown格式
他没有wordpress那样独立的管理后台
因此没法做到随时随地发文，必须在部署的那台电脑上上传

说了这么多，都只是梗概
写一篇博文可能会涉及具体的操作步骤
有问题，欢迎讨论

##何其谓文明

高中便读过了反乌托邦三部曲
在我内心，也时常会将乌托邦与共产主义化为一列
当今中国的意识形态教育就和美丽新世界中的有所类似
细细想来
那样的文明也是幸福的
人生来就是一个被不断洗脑的过程
所谓快乐，幸福，爱情诸如此类
都是慢慢沉淀的产物

人类就是这样前赴后继
一代比一代伟大的

其实，并不觉得，如今是最美好的时代
for one thing
我们付出了太多
追赶别人是一件辛苦的事情
徐徐前行难道不是一件好事吗
人类的存在都是一个谜
难道人活着不应该是探索自己吗

北欧国家曾经批评过中国
为什么这个国家敢发展得这么快
他难打不为自己的人民着想吗
他们难道不痛苦嘛

说的很对啊
一味地devote，往往会造成内心空虚的场景
这样倒不如乌托邦
至少大家的内心是充实的
阿尔法、贝塔、伽马、德尔塔和埃普斯隆各司其职
这个世界进步了
大家还很开心

跳出来吧，这些都也止于思考了
或许
唯物史观是对的

谈谈人类吧
先从水谈起
看起来平淡无奇的东西
确实这个世界存在的原因

按理说，地球是不可能存在水的
就地球的位置而言
水金地火木土
离太阳越近，行星应该都是岩石星球
离太阳越远，就应该是气态的
而谁哪来的呢？

曾经有好多机构分析过
水中含有两种不同的物质
就类似于碳的有同素异形体这种
而且这两种形态产生的条件都是不同的
换句话说
就以为止水并非地球之物
那么或许人类也不是

水还有许多神奇的地方
最先想到的便是雪花了
精妙绝伦
而且，世界上没有两片相同的雪花
曾经有一个叫宾利的人（被誉为雪花男人）
一生都在拍摄雪花
他发现任何两片雪花总是不一样的

更匪夷所思的还有
水变成冰的密度降低了
听起来好像没什么作用
但想象一下，生命是存在于水中的
那么当类似于寒武纪这样的冰河期到来时
水结成冰并不会沉下去
还是会悬浮在水上面
阳光依旧可以照射到下面的说中
生命也得以延续

让人捉摸不透的太多
甚至有说法
谁才是人类最终的能源
因为他可以反反复复
我初中就曾设想过
会不会有这么一个条件
水中氢氧共价键断裂
产生能量再放出水
如此反反复复
那岂不是人类太空旅行
只要一桶水就足够了呢

这个假设并不是没有道理的
As we know
地球上存在过恐龙
那么恐龙到底是怎么灭绝的呢
陨石撞地球？
那么陨石坑又在何处
唯一的解释便是
一个陨石砸到了海里
触发了水爆炸（产生能量的机制）
反反复复
爆发出毁灭地球的爆炸
这一点也可以从氢弹爆炸的蘑菇云和火山爆发的蘑菇云一样来解释

这一点也在人类未解之谜（终神之地，图拉遗迹）也有所显示
在其中的石像中
有石像手中拿着水枪一样的武器
或许水本就是人类能源问题的最终解

匪夷所思的太多
暂时先到这吧，下回再思考。

-钢铁之心，难抉得失

唐尼的心事：
“漫威系”即将迎来众超级英雄及反派的大婚，而最为苦恼的则是“钢铁侠”唐尼，因为当前至少有三对新人会相继举办婚礼（虽然有对外公布已举办婚礼的情况，但是举办多场小型婚礼派对在国外很常见），而后续档期非常紧缺的唐尼则可能面临抉择尬境，这究竟是什么原因呢？

一、参加凤凰女的婚礼：
理由：同宗族共血脉，超级英雄代表（索菲·特纳）：
作为《权利的游戏》中，北境临冬城的史塔克一族，珊莎·史塔克（索菲·特纳饰）与漫威电影宇宙的钢铁侠托尼·史塔克同属史塔克家族，家族血脉从古至今延续至遥远的未来，因此作为同宗，唐尼也需要赏脸索菲·特纳的婚礼；
虽然珊莎·史塔克曾被迫嫁给“小恶魔”提利昂·兰尼斯特，但是一日夫妻百日恩，而“小恶魔”（饰矮人王）在《复仇者联盟3》中起到了非常关键的作用，矮人王一方面为复仇者三巨头中的雷神打造了暴风战斧，另一方面虽然雷神没打败灭霸，但是钢铁侠托尼·史塔克还是一定程度上通过研究灭霸残旧的无限手套打造了属于复仇者联盟的专用无限手套，最终复活了所有人并且打败了另一时空的灭霸，因此钢铁侠也应该会给矮人王（小恶魔）的妻子一个面子；
我们都知道当前漫威电影宇宙中最强的是惊奇队长，而索菲·特纳接下来在漫威的《X战警-黑凤凰》（由二十世纪福克斯公司出品）中饰演黑凤凰一角，而在漫威漫画故事中，黑凤凰又被称为“惊奇女孩”，一度强大到实力接近无限，可见其实力，而X战警的角色回归漫威电影宇宙也是影迷们呼喊声非常高的一件事，今后凤凰女与复仇者联盟的交际会更多，作为钢铁侠的唐尼还是要捧一下场的。

二、参加勇·罗格的婚礼：
理由：出生入死相随，培养惊奇队长（裘德·洛）：
福尔摩斯与华生是生死相交的好友，没有华生这样出色的队友，也就没有福尔摩斯这样的传奇故事，在真实工作中，裘德·洛作为第36届土星奖最佳男配角，在《大侦探福尔摩斯》中与唐尼配合献上了精彩的互动演出，为福尔摩斯的成功奠定了坚实的基础，并且唐尼在《复仇者联盟4》之后会出演《大侦探福尔摩斯》的续集，俩人还要继续搭档来破解疑案，虽然在《大侦探福尔摩斯》中，福尔摩斯是不希望好兄弟因为结婚受到约束的，但是在真实生活中，我们的唐尼是不是一定要参加和祝福我们的裘花呢？
在漫威的《惊奇队长》中，裘德·洛饰演的勇·罗格曾是惊奇队长的教练（不称为导师是因为不具备导师的真心），如果没有勇·罗格便没有惊奇队长，没有了惊奇队长，宇宙魔方就不会在很长一段时间内受到噬元兽和神盾局的保护，《复仇者联盟4》中钢铁侠也不会在太空中获救，最终灭霸也不会因为钢铁侠的牺牲而被打败，因此我们的唐尼是不是要送去双重祝福给我们的裘花呢？

二、参加星爵的婚礼：
理由：迪士尼新巨星，复联四创造者（克里斯·帕拉特）：
我们都知道，逆转无限的根本不是什么无限宝石，也不是什么皮姆粒子，真正逆转整个漫威电影宇宙的是“米老鼠”迪士尼，作为近年来迪士尼力捧的影星，克里斯·帕拉特每次都交出了优异的答卷，在《银河护卫队》以及《侏罗纪世界》两部系列的电影中都取得了超乎预计的票房收入，因此克里斯·帕拉特今后的星途一定是坦荡无疑的，并且现在的岳父是美国好莱坞所在地加州的前州长，况且还是能够随时穿越时空终结反派“终结者”，既然我们的星爵如此前途无量，迪士尼旗下的漫威电影宇宙领衔人物钢铁侠唐尼也值得参加我们星爵的婚礼吧？
在《复仇者联盟3》中，因为发现“岳父”灭霸为了无限宝石牺牲了自己的挚爱，因此不顾队友们的缜密计划，将灭霸打醒，也造成了差点被小蜘蛛拿到手的无限手套被灭霸夺回，此后灭霸更是势如破竹地消灭了宇宙一半的生灵，最终才发生了《复仇者联盟4》的故事，因此讲星爵创造了《复仇者联盟4》也不为过，当然如果没有星爵，我们又怎么能再多看一次唐尼呢？

大家觉得唐尼该怎么安排行程，又该分别为新人带去怎么样的祝福呢？
